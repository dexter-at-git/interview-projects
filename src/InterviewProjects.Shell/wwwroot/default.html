Image search

Single-page application to search image using Google API with ability to save search history in local storage.

Technologies and tools that were used:

  • AngularJS as a main framework to build SPA
  • Jasmine as a framework for unit testing
  • Bower as a package manager
  • Grunt and gulp as a build tools (just an example to solve different tasks)

Transcipher

Single-page application to encrypt or decrypt text by selected algorithm.
Implemented encryption to and from Morse code. Solution allowed easily extension with another encryption and decryption algorithm.

Technologies and tools that were used:

  • ASP.NET Core as a backend framework
  • xUnit as a framework for backend unit testing
  • AngularJS as a framework to build SPA
  • Jasmine as a framework for frontend unit testing
  • Bower and Nuget as a package managers
  • Grunt and gulp as a build tools (just an example to solve different tasks)

Sms Manager

Single-page application to send sms messages, browse sent mesages and view statistics.

Technologies and tools that were used:

  • ASP.NET Core as a backend framework
  • Entity Framework Core as ORM framework
  • SQLite as a relational database
  • xUnit as a framework for backend unit testing
  • AngularJS as a framework to build SPA
  • Jasmine as a framework for frontend unit testing
  • Bower and Nuget as a package managers
  • Grunt and gulp as a build tools (just an example to solve different tasks)
